# Kampánus Rakovník - Static Website

Web: https://kampanusrakovnik.cz

### Build
```
    make init
    make build (make run-tests, assets, generate, static)
```
Static HTML files generated into `/public`

### Development
```
    make init
    make build (make run-tests, assets, generate, static)
    make local
```

Web available at http://localhost:8000
